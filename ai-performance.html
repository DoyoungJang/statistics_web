Interpretation

입력값 가이드

  • Accuracy, Sensitivity, Specificity, NPV는 proportion 기반 비열등성 계산으로 처리합니다.
  • AUC는 Hanley-McNeil 분산 근사를 사용하며 one-sided alpha 0.025 기준으로 양측 95% 신뢰구간 해석을 함께 맞춥니다.
  • F1-score, mAP, R-squared는 bootstrap 또는 파일럿 데이터 기반 SD를 이용한 mean 근사식으로 처리합니다.
  • FP per Image, MAE, RMSE, MAPE는 lower-better 지표로 계산합니다.
  • Bland-Altman은 paired difference의 평균과 SD를 이용해 limits of agreement의 신뢰구간이 허용 차이 안에 들어오도록 계획합니다.
  • Measurement Accuracy는 허용오차 내 비율을 accuracy로 정의한 경우를 가정합니다.

Formula

적용 공식

Proportion Metrics

H0: pAI ≤ pRef - ΔNI, n = (z1-α × √(p0(1-p0)) + z1-β × √(p1(1-p1)))² / (p1-p0)²

Mean Metrics, Higher Is Better

H0: μAI ≤ μRef - ΔNI, n = ((z1-α + z1-β) × σ / (μ1-μ0))²

Mean Metrics, Lower Is Better

H0: μAI ≥ μRef + ΔNI, n = ((z1-α + z1-β) × σ / (μ0-μ1))²

AUC

Hanley-McNeil variance approximation with fixed positive/negative case ratio

Bland-Altman

Upper LoA = |d̄| + 1.96s, SE(LoA) ≈ √(3s² / n), n ≈ 3((z1-α/2 + z1-β) × s / (Δ - |d̄| - 1.96s))²

Assumptions

주의사항

계산식은 규제기관이 직접 고정한 공식이 아니라, benchmark와 비열등성 마진을 두고 one-sample non-inferiority 문제로 실무적으로 구현한 표준 근사식입니다.

AUC, F1-score, mAP, R-squared처럼 집계형 metric은 분산 추정 방법에 따라 샘플수가 달라질 수 있습니다. 최종 제출 전에는 파일럿 데이터 기반 bootstrap 또는 통계분석계획서(SAP) 수준의 검토를 권장합니다.

Glossary

AI 성능평가 샘플수 전문 용어 설명

AI 성능 샘플 수 계획에서 자주 등장하는 성능지표와 분석 용어를 한국어 기준으로 정리했습니다.

Sensitivity

실제로 질환이나 병변이 있는 경우를 AI가 양성으로 잘 찾아내는 비율입니다. 놓치면 위험한 기능에서 특히 중요합니다.

Specificity

실제로 질환이 없는 경우를 AI가 음성으로 올바르게 분류하는 비율입니다. 불필요한 추가 검사나 false positive 부담과 관련됩니다.

PPV / NPV

양성예측도와 음성예측도입니다. AI가 양성 또는 음성이라고 했을 때 실제로 맞을 확률을 뜻하며, 질환 유병률의 영향을 많이 받습니다.

AUC

ROC 곡선 아래 면적입니다. threshold를 바꿔가며 전반적인 분류 성능을 보는 지표로, 1에 가까울수록 좋습니다.

F1-score

Precision과 sensitivity를 함께 반영하는 조화평균입니다. 양성과 음성 비율이 불균형할 때 분류 성능 요약 지표로 자주 씁니다.

mAP

객체 검출에서 쓰는 대표 지표입니다. 여러 threshold에서 precision-recall 성능을 평균내어 탐지 품질을 평가합니다.

Dice Score (DSC)

분할 결과와 정답 영역이 얼마나 겹치는지를 보는 지표입니다. segmentation에서 가장 흔히 쓰는 합치도 지표 중 하나입니다.

IoU

예측 영역과 정답 영역의 교집합을 합집합으로 나눈 값입니다. segmentation과 detection 모두에서 겹침 정도를 평가할 때 사용합니다.

ICC

Intraclass Correlation Coefficient의 약자로, 측정값이 얼마나 일관되게 재현되는지 보는 지표입니다. measurement 기능에서 자주 사용됩니다.

Bland-Altman

두 측정 방법의 차이를 평균 차이와 limits of agreement로 보는 방식입니다. 단순 상관계수보다 실제 측정 일치도를 평가하는 데 더 적합합니다.

MAE / RMSE / MAPE

측정 오차를 요약하는 대표 지표입니다. MAE는 평균 절대오차, RMSE는 큰 오차에 더 민감한 제곱근 평균제곱오차, MAPE는 백분율 오차입니다.

Lesion-level / Patient-level

분석 단위가 병변 하나인지 환자 전체인지 구분하는 개념입니다. detection에서는 이 단위 차이가 샘플 수와 endpoint 해석에 큰 영향을 줍니다.

False Positives Per Image

한 이미지당 AI가 잘못 표시한 탐지 개수입니다. detection AI의 임상적 부담을 보여주는 중요한 보조 지표입니다.

Latency

입력 후 AI 결과가 화면에 나타나기까지 걸리는 시간입니다. 실시간 AI에서는 정확도만큼이나 중요한 성능 요구사항입니다.
