# 의료기기 인허가 통계 계산기

의료기기 인허가 문서 작성 시 필요한 통계 계산을 빠르게 수행하기 위한 정적 웹 애플리케이션입니다.

## 현재 포함 기능

- 단일 비율 추정용 최소 샘플수 계산
- 두 비율 비교용 최소 샘플수 계산
- 두 평균 비교용 최소 샘플수 계산
- 예상 탈락률 반영 모집 목표수 계산

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 계산 유형을 선택합니다.
3. 가정값을 입력하고 결과를 확인합니다.

## 기본 가정

- 독립 표본
- 양측 검정
- 두 군 비교 시 1:1 배정
- 두 평균 비교 시 두 군의 분산이 유사함

## 주의

비열등성, 동등성, paired design, 생존분석, ROC 분석 등은 별도 공식을 추가로 구현해야 합니다.
