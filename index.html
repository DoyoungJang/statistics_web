최소 샘플수 계산

시험 목적에 맞는 계산 유형을 선택하고 가정을 입력하세요.

단일 비율 추정

민감도, 특이도, 성공률처럼 하나의 비율을 목표 정밀도로 추정할 때 사용합니다.

값을 입력하고 계산하면 결과가 여기에 표시됩니다.

두 비율 비교

대조군과 시험군의 성공률, 반응률, 사건발생률 차이를 검정하려는 경우에 사용합니다.

값을 입력하고 계산하면 결과가 여기에 표시됩니다.

두 평균 비교

연속형 지표의 평균 차이를 검정할 때 사용합니다. 두 군의 분산이 비슷하다는 가정을 둡니다.

값을 입력하고 계산하면 결과가 여기에 표시됩니다.

Interpretation

입력값 가이드

  • 예상 비율은 선행연구, 파일럿 시험, 유사 제품 결과를 기반으로 설정합니다.
  • 허용 오차는 추정 정밀도입니다. 예를 들어 0.10이면 ±10% 폭을 의미합니다.
  • 검정력은 보통 80% 또는 90%를 많이 사용합니다.
  • 예상 탈락률을 입력하면 계산된 최소 샘플수와 모집 목표수를 함께 확인할 수 있습니다.

Formula

적용 공식

단일 비율 추정

n = z² × p × (1 - p) / d²

두 비율 비교

n = (zα/2·√(2p̄(1-p̄)) + zβ·√(p1(1-p1)+p2(1-p2)))² / (p1-p2)²

두 평균 비교

n = 2 × (zα/2 + zβ)² × σ² / Δ²

Regulatory Note

주의사항

본 계산기는 독립 표본, 양측 검정, 동일 배정비를 기본 가정으로 합니다. 비열등성, 동등성, paired design, 생존분석, ROC, sensitivity/specificity 목표 하한 검증 등은 별도 공식을 적용해야 합니다.

민감도와 특이도 평가 시험은 양성군과 음성군 샘플수를 각각 따로 계산한 뒤, 실제 모집 가능 비율을 고려해 전체 모집계획으로 전환하는 것이 일반적입니다.
